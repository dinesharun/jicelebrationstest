JASMIN 15 Anniversary Web Portal Usage Guide

Please consider the following points when using the registration portal
  1. When logging the first time you will be asked your fullname and email-id, kindly fill the correct details.
  2. For registering for any single event just go to that event and click the "Register for this event" button
  3. For registering for any group event just go to that event and fill an appropriate group name and click the "Register for this event" button
  4. For registering for any group event in case, if your partner has already registered fill the same group name and click the "Register for this event" button
  5. To unregister from any event, kindly press the "Unregister from this Event" button in the particular event.
